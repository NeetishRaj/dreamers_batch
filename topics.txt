Arrow function syntax
    Few syntax concentions
        1. If single return statment then that return statement with curly braces can be ommitted.
            Example: 
            const add = (num1, num2) => {
                return num1 + num2;
            }
            const add = (num1, num2) => num1 + num2;

        2. If single argument then parenthesis can be ommitted.
            Example:
            const square = (num) => num * num;
            const square = num => num * num;

Q. What are the differences between for loop and Array.forEach loop ?
A.
    1. 
        forEach loop will execute for all items of the array ALWAYS but for-loop can be discontinued ealier.
        We cannot break from forEach loop in between the loop but we can for for-loop.


    2. 
        forEach loop will not modify the array content while iterating over it.
        but using for-loop we can modify the array content while iterating over it.

        const even = new Array(200);

        for (let i = 0; i < even.length; i++) {
            even[i] = (i + 1) * 2;
        }

        // This will update the even array

        even.forEach((item, index, arr) => {
            even[index] = (index + 1) * 2;
        })

        // But this will not update the even array

    3. 
        break doesn't work inside forEach loop
