JS part7: Objects
#################

Reference: https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Object

Objects
    syntax of an object: 
        {
            key1: value,
            key2: value,
            ..
            ..
        }

        Here key1 and key2 are supposed to be valid identifiers or unique strings

    Basics
        ways of accessing properties in an object
            1. Dot notation
                const obj  = {name: "john", age: 29}
                obj.name  // john

            2. [] square bracket notation.
                const obj  = {name: "john", age: 29}
                obj["name"] // john

    Object class
    
Object properties and methods
    Object.keys()
    Object.values()
    Object.entries()
    Object.hasOwnProperty()

Q. When to use square brancket notation to access Object properties in JS?
A.
    We use square brackets When
    
    The property name is not a valid identifier ( i.e has spaces, special characters) 
    Example:
        const obj = {
            "some prop name": 345
        }
        obj["some prop name"]  // 345


Q. Guess the output.

    const obj = {
        data: [
            {
                name: "rahim",
                age: 23,
                address: {
                    city: "jaisalmer"
                }
            },
            {
                name: "sugandh",
                age: 22,
                address: {
                    city: "Indore"
                }
            }
        ]
    }

    console.log(obj.data[1]["address"]) // {city: "Indore"}
    console.log(obj.data[1]["address"].city) // Indore
    console.log(obj.data[1]["address"]["city"]) // Indore
