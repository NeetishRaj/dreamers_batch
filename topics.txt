CRUD

    C - Create (HTTPS POST method)

    U - Update (HTTP PUT method)

    D - Delete (HTTP DELETE method)


Setup a simple HTTP Server with json-server

    - npm install -g json-server
    - Create a server folder
    - create a db.json file with initial data.
    - run `json-server --watch db.json`
